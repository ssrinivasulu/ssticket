Schema Browser

Schema Browser - Oracle Databases

With the Schema Browser you can comfortably navigate through the objects defined in the database. The database objects are always displayed under the currently active Connection. This is selected either via the Connect menu and selection of the connection, or it can be set automatically upon activation of particular SQL Worksheet or Object Detail dialog.

Category filter

Show/Hide Categories option is available via the button in the bottom-right of the Schema Browser. When clicked, Categories sheet opens.
You can select which object categories will be displayed in the Schema Browser (for example, objects outside of your schema are not needed).

Schema Browser filter

Items can be filtered; you can use wildcards in object names.

Bookmarks

You can set bookmarks for your favourite database objects to have a quick access to them. Bookmarks are displayed in a separate section.

Running Stored Procedures and Functions from Schema Browser

To run a Stored Procedure or a Function with parameters, right click the object in the Schema Browser and select Run or Run with...

For details, see Run Stored Procedures and Functions.

Schema Browser - PostgreSQL

Schema Browser for PostgreSQL shows PostgreSQL specific items like Collations, Catalogs, Databases etc.

To set the Active Schema, select Set active schema from its context menu.

The Schema Browser provides standard functions for each database object (e.g. rename, edit, drop).

When you select a database object, the object properties are displayed in the Object Detail dialog. If the Object Detail dialog is closed or hidden, click on the selected database object to open the dialog.

Schema Browser - MySQL

Schema Browser for MySQL shows a list of databases and specific MySQL objects.

Active Database

To set the Active Database, select Set active database from its context menu.

Standard Features

Schema Browser provides standard functions for each database object (e.g. rename, edit, drop).

Database and User Creation

In the section Databases you can easily create a new MySQL database.
Define a database name, set a character set and collation values and click OK to create the database.
New users can be created via the Schema Browser as well.
Specify a new user name and password and click OK to add the user to the active database.

Object Detail

When you select a database object, the object properties are displayed in the Object Detail dialog. If the Object Detail dialog is closed or hidden, click the selected database object to open the dialog.
