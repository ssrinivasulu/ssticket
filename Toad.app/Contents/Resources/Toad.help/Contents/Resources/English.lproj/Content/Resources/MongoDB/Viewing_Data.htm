Viewing Documents in Object Detail and Detail Pane

Learn how data viewing works in MongoDB.

Object Detail

Double-click a Collection in Object Explorer to open it in Object Detail.

In Object Detail, you can view Documents in a Collection in two ways:

  • Grid - data is clearly arranged in a grid and can be sorted using columns
  • Json - data is displayed in raw format
  • Indexes - displays information about Collection Indexes
  • Stats - displays Collection metadata

Regardless of the used view, Documents can be filtered using Outline View. See Filtering data using Outline View for more infomation.

Object Detail buttons
Button Option Description
Refresh Refreshes Documents in Grid.
Show/Hide Detail Pane Shows/hides all instances of Detail Pane.

Detail Pane

Detail Pane is a separate window, which displays Field values for a document that is currently selected in Outline View.

To open Detail Pane:

  • Check Detail for any Field in Outline View. Each Field with all its sub-fields is displayed in a separate Detail Pane.


  • A new Detail Pane windows opens. It will display the Field value (if the Field has sub-fields, their values will be displayed as well)


Detail Pane buttons
Button Option Description
Stack Vertically Stacks opened Detail Panes next to each other.
Stack Horizontally Stacks opened Detail Panes below each other.
Show in Tabs Each Detail Pane will be shown in its own tab.
Switch Rows/Columns Switches Fields/Documents.
Close Detail Closes Detail Pane.
