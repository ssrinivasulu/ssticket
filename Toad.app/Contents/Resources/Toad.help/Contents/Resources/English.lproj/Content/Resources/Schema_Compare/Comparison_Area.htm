Comparison Area

Review specific differences and choose the action to resolve them.

This part of Schema Compare utility displays specific differences between code of database objects.

Note: Hover over a difference icon to see more information about the difference.

Comparison Area Toolbar

UI Element Option Description
Mark as Important Marks selected difference as important to easily distinguish it from other differences.
Add to Excluded Adds selected difference to excluded list. Differences in this list can be shown/hidden using a button on Overview Grid Toolbar.
Previous Change Switches to the previous difference.
Next Change Switches to the next difference
Previous Unread Change Switches to the previous difference marked as unread ().
Next Unread Change Switches to the next difference marked as unread ().
Export Script Exports the database object code as a SQL file.
Copy Copies the database object code to clipboard.
Open in SQL Worksheet Displays the database object code in SQL Worksheet.
Choose Action

Indicates which action should be done in case of a particular difference.

  • Item should be changed in left - Object in source schema will be replaced by object in target schema.
  • Item should be changed in right - Object in target schema will be replaced by object in source schema. This is the default action and is represented by a blank space in the Overview Grid.
  • Item should be changed in both - Object will be replaces in both source/target schema with another object.
  • I have no idea - No action.
Deploy Deploys object from source schema into the default schema of target database. Applicable only for "Only in source" differences and when comparing schemas of different connections.
