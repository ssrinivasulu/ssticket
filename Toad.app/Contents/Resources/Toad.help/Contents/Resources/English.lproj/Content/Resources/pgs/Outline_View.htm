Outline View

Have Outline View always in sight to see the object's structure.

Outline View has two modes:

  • Code Outline
  • Database Object Outline

Code Outline

The Outline View displays the logical structure of PL/SQL objects displayed in Stored Procedure Editor.

Clicking on any node in Outline View will transfer your cursor to the corresponding part of the code.

Database Object Outline

Outline View also shows all important details of the displayed database object in active Object Detail/Describe Object window.

By clicking on any item in Outline View you can navigate to the corresponding object in Object Detail/Describe Object window.
