Create New Type

Use wizard to create a new type easily.

Note: This feature is available only for Oracle and PostgreSQL databases.

Right-click Types in Object Explorer and choose New Type.

Oracle

Choose the new Type Schema and enter its Name.

Define a new type and click Finish. The created type opens in a new SQL Worksheet.

PostgreSQL

Choose the new Type Schema and enter its name. Also choose the Type.
