Create New View

Use wizard to create a new View easily.

Right-click Views in Object Explorer and choose New View or click the New Object button on Object Explorer toolbar.

Define View Schema and Name (additionally Restriction for PostgreSQL and Oracle). Click Next to continue.

In the next dialog Add objects that should be part of your View.

If your View contains columns from multiple database objects, you can Join them in this dialog.

From the previously chosen objects pick columns that should be selected in your View.

You can define conditions for any previously chosen columns.

In the last dialog you can choose columns by which the new View will be ordered.
