About Schema Compare

Comparing objects between databases has never been easier.

Note: This feature is available for Oracle databases only.

With Toad, you can now compare database objects of two schemas.

Schema Compare tool can be opened in two ways:

  • In Object Explorer, select any two schemas, right-click and choose Compare Schemas. This method is recommended when comparing two schemas of the same database.


  • Go to Database Menu and select Schema Compare. This opens Schema Compare Wizard. Specify Source and Target schemas and select the desired Object Types and Comparison Rules. Once you're done, click Compare.



Wait until the comparison process is completed. This may take a while depending on the number of objects compared. Once the comparison is done, you can see all the differences between your schemas and perform various actions.

Schema Compare utility consists of three parts:

  • Overview Grid - displays differences which can be filtered, you can also mark differences as read/unread/important/excluded
  • Comparison Area - shows the specific differences between database object in source/target schema.
  • Outline View - contains settings which allow you to control which types of differences should be considered/ignored.
