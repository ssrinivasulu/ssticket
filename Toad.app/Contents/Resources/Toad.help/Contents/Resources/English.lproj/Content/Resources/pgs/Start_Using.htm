Start Using Toad

Find information about how to install and start working with Toad.

To install and start Toad :

  1. Open App Store application.
  2. Search for "Toad".
  3. When Toad is found, click the Get button, it changes to INSTALL APP, click it and installation starts.
  4. Confirm the installation, review License Agreement, confirm installation location and authorize your installation with Administrator credentials.
  5. "Toad" application is installed in Applications and application icon is added in the Launchpad.
  6. Start Toad from Applications or from Launchpad.

First start

Welcome dialog opens on the first start of Toad on your machine after the first installation, not after upgrading from previous versions.

Welcome dialog

The dialog allows a user to set which database platforms will be used. The setting influences database platforms offered for connections etc.

Note: Oracle is set as default database platform (this can be changed in Toad Menu | Preferences | General | Preferred database platform.

To enable PostgreSQL and MySQL databases as well, check their checkboxes. The databases can be enabled/disabled at any time in Toad Menu | Preferences | Databases.

Oracle, MySQL and PostgreSQL JDBC driver are distributed with Toad and pre-set: ojdbc7.jar for Oracle and the latest drivers for both MySQL and PostgreSQL.

Caution: If you don't have Oracle Client installed on your machine, then you can download it from the Oracle website, or download a different Oracle JDBC driver, if necessary.

Additional information about installing Oracle Client can be found on Oracle documentation site.

Used JDBC drivers can be changed in Toad Menu | Preferences | Databases. However, it is strongly recommended to use the default drivers distributes with the application.
