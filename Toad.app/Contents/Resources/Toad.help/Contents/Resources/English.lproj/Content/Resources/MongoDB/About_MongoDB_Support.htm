About MongoDB support

What can you do with MongoDB?

Toad - Mac Edition currently supports MongoDB with focus on viewing, finding and filtering relevant data quickly and clearly.

Viewing MongoDB database objects

Similar to other databases, MongoDB objects can be viewed in Object Explorer. See more information.

Viewing Documents in a Collection

Documents can be seen in Object Detail of a Collection. You can view data in Grid, or in raw JSON format. See more information.

You can also view one Document at a time in Detail Pane, which displays Fields selected in Outline View.

Filtering data using Outline View

Outline View displays estimated Document structure (which is updated dynamically). By checking/uncheckingFields, you can show/hide them in Object Detail. Exists and Filter fields are used for finding specific Documents. See more information.

Executing Mongo shell commands via Worksheet

Toad also allows you to execute various Mongo shell commands in Worksheet. See more information.

Exporting/Importing JSON data

Exporting/Importing data from/to your Collections is easy, simply right-click a Collection and choose Export/Import Collection.
