Create New Table

Use wizard to create a new database table easily.

To create a new table:

  • Connect to your database
  • Right-click Tables in Object Explorer and choose New Table.

or

  • Click the New Object button on Object Explorer toolbar.

Define the Table name, Table Type and click Next to continue.

In the following dialog create the Table columns by clicking New... button. To create a Foreign Key, click the New FK... button. Once you're done creating columns and keys, click Finish to create your Table.

Note: At any time during the wizard you can click the SQL button to display preview of the SQL code of your Table.
