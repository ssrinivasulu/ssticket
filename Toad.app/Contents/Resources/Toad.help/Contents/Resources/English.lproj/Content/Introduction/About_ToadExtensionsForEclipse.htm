About Toad™ Extension for Eclipse

Toad Extension for Eclipse is an Eclipse plugin for Oracle, PostgreSQL and MySQL databases. It extends the functionality of Eclipse by options for Oracle, PostgreSQL and MySQL databases and offers various database management features.

Benefits

Toad Extension for Eclipse allows you to:

  • connect to your Oracle, PostgreSQL or MySQL database, view, explore and edit database structure, database objects and properties
  • manage database objects, easily add, edit or drop objects
  • manage data stored in your database, add, edit or remove records
  • write complex PL/SQL code comfortably in Editor. Available features depend on target database platform.
  • get detailed information about database objects - Explain plan, SQL profiler, Quick Doc, Oracle parameters etc. Features depend on target database platform.
  • work with SQL statements from within Java Editor

Toad Extension for Eclipse plugin views

  • Connections — Overview and selection of configured database connections
  • DBMS-Output * — Overview of all outputs from DBMS_OUTPUT
  • Dependencies ** — Dependencies of the object and on the object
  • Deployment Log — Displays deployment history of the session
  • Editor — Editing, running, storing and compiling of Stored Procedures, Views, Triggers, Packages and Types
  • Object Explorer — Allows you to browse through the objects of your database
  • Object Detail & Describe Object — Describe the selected database object and allow
  • Oracle Parameters * — Displays Oracle connection parameters and settings
  • Outline- Shows the logical structure of a database object
  • Quick Doc— Shows the PL / Doc of the selected database object
  • Run Stored Procedure — Displays results of PL/SQL queries execution
  • Session Browser * — Sessions related to logged user
  • SQL Monitor — Log of all executed SQL Statements
  • SQL Problems — Displays issues in run queries
  • SQL Recall — History of executed SQL statements
  • SQL Statements *** - Displays information about SQL statements within Java code
  • Search Dialog — Database Search pane- Browse the source code of stored procedures
  • Parameters * — Overview of the database and session parameters
  • Worksheet — Execution and committing of SQL statements


* View is available for Oracle databases.

** View is available for Oracle databases and experimentally on PostgreSQL databases.

*** View is available when Java editor integration plugin is installed.
