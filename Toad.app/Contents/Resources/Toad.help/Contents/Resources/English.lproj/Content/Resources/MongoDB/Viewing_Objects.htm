Viewing objects in MongoDB database

Learn more about Databases and Collections.

Object Explorer currently displays the following MongoDB objects:

  • Databases
  • Collections
  • Indexes
  • Functions (JavaScript functions used for MapReduce operations)

Note: You can Export and Import Collections using the right-click menu. If you are exporting, make sure you have enough space on your local machine, collections can be extremely large.

You can also create a new Database/Collection using button on Object Explorer toolbaror by right-clicking a Database/Collection and choosing Create New Database/Collection.

Creating a new Database

Simply enter new Database Name and click Finish.

Creating a new Collection

Name is the only mandatory field to create a new Collection, other settings are optional:

  • AutoIndexId - when enabled, automatically creates index on _id field.
  • Capped Collection - limits Collection size (by limiting its file size, Document number, or both). The Collection oldest entries are automatically overwritten upon reaching its maximum size
  • Use PowerOf2Sizes - when enabled, each record has allocated space as a power of 2 (e.g. 64 KB)

Creating a new Function

You need to choose the target Database, the Function Name and you can also enter its Code right away or edit it later in Object Detail.
