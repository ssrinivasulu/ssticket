Search Dialog

Find the objects you need using one of several search methods.

The Search dialog allows you to comfortably search for database objects. Wildcards are supported and you can also enable/disable case sensitivity or restrict search to particular schemas or object types.

Tip: The fastest way to open a database object by its name is by using Open Quickly dialog (ALT+COMMAND+O). You can search for a specific object by name, prefix, or pattern (wildcards are supported). The dialog then displays all matches found in all databases/schemas in your Connection.
