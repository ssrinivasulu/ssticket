Data Export

Export your data easily into various formats.

You can Export data from both Object Explorer and from Data Grids (Object Detail, Describe Object, Result Set).

Exporting data from Object Explorer

Object Explorer allows exporting data to the following working formats in order to utilize Import:

  • SQL- as INSERT or MERGE statements
  • CSV

To export data from Object Explorer:

  1. Right-click one or more selected tables in Object Explorer and choose Export Data...

  2. Export Data wizard opens. Now you can choose additional tables for export or add child and parent tables of the already chosen ones.

  3. In the next dialog choose the exported data format, the Line Terminator, Destination and additional settings.

  4. You can exclude specific data type columns from the export or Limit Number of Records.
  5. Or you can exclude specific columns from the export by unchecking them in this dialog.

  6. The final dialog contains preview of your exported data. You can change SQL Options or CSV Format Settings, depending on the chosen export format.

  7. Once you're done, click the Finish button to export the data.

Exporting data from Data Grid

Data can be also exported from Data Grid, which is part of Object Detail, Describe Object and Result Set windows. The data can be based on Grid Selection or Query Result.

The supported data export formats are:

  • SQL - as INSERT or MERGE statements
  • CSV
  • HTML and XML for Query Results

To export data selection from Data Grid:

Grid Selection - Select one or more rows from any Data Grid.

Query Result - Execute a query that returns at least one row. Records displayed in Data tab of Object Detail/Describe Object are considered a query result as well.
