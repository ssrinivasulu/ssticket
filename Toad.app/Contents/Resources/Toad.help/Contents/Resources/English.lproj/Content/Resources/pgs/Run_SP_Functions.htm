Run Stored Procedures and Functions

Find out how to run Stored Procedures, Functions etc.

You can Run Procedures and Functions stored in your database right in the Object Explorer. Simply right-click a Procedure/Function and choose Run.

When you Run a Procedure/Function, you will have to enter its parameters in Set Input Parameters window. Then click the Run button, and once the Procedure/Function is finished, results will be shown in Run View. If your Procedure/Function requires no arguments, Run View will show you results immediately.

If you have run more Procedures/Functions, you can switch between their results using dropdown arrow of Show Results button.
