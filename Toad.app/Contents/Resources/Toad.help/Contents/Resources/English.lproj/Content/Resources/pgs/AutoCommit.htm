AutoCommit

Save DML changes made to your database automatically.

AutoCommit refers to automatically committing all changes you make in your database using certain DML (Data Manipulation Language) statements - INSERT, ALTER, UPDATE, DELETE...

Note: Enabling AutoCommit disables Commit/Rollback Transaction buttons in certain windows.

Toad allows you to enable/disable AutoCommit for each database in two ways:

  • Connection Properties - check/uncheck Enable AutoCommit option when creating a new connection or editing an existing one.

  • Toad application toolbar - go to Database menu and Enable/Disable AutoCommit.
