Management of BLOBs

View, export and import BLOBs.

Toad allows you to work with BLOBs - you can open them in application of your choice and also import/export them.

View content stored in BLOBs

To open a document or a file stored in your BLOB field, right click the appropriate cell in Object Detail/Describe Object | Data Grid and choose Open or Open With.

The application used to open a BLOB depends on the specific Open option:

  • Open and Open | System Editor - default system application is used to open the BLOB
  • Open With | Text Editor - BLOB is opened in plain text editor

Setting Default BLOB Viewers

It is possible to set BLOB Viewer via Toad Menu | Preferences | Grid Options | BLOB Viewer section (see Settings for more information).

Export and Import

  • To export BLOB content to a file, right-click the appropriate cell in Data Grid and choose Export BLOB to File...
  • To import a file into a BLOB, right-click the appropriate cell and choose Import BLOB from File...

    		•

    Note: When you import a file into a BLOB cell in Data Grid, the previous file in the BLOB will be overwritten. In case you accidentally overwrite a BLOB, you can Rollback the change (unless you have enabled AutoCommit).
  • To clear the BLOB content, right-click the appropriate cell and choose Set Null.
