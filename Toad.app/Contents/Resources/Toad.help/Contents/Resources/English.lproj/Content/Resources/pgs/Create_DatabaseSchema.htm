Create new Database/Schema

How to create Database/Schema in all databases.

This guide describes the process of creating a new Database/Schema or equivalent objects for the following database platforms:

  • Oracle
  • PostgreSQL
  • MySQL

Oracle Schema/User

Right-click any Schemas node in Object Explorer and choose New Schemaor right-click Users in Object Explorer and choose New User.

Enter the new Schema/User Name, Password and Confirm Password. Also don't forget to choose the desired Default and Temporary Tablespaces.

In the next dialog choose which roles should be granted to the new Schema/User.

The Schema/User system privileges can be set in the next dialog.

If you need, you can also set Tablespace Quotas for the newly created Schema/User.

PostgreSQL Database/Schema

Right-click Schemas/Databases in Object Explorer and choose New Schema/Database.

The new Schema/Database wizard is similar. Choose the object Owner and enter the object Name.

When creating a new Database you can proceed to the next dialog and choose your new Database Template, Tablespace and set other settings - Encoding, Collation, Character Type and Connection Limit.

MySQL Database

Right-click Databases in Object Explorer and choose New Database.

Specify Database Name and additionally Character Set and Collation, if needed.
