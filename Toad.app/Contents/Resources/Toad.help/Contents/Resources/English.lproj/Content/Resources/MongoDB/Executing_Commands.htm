Executing Mongo shell commands using Worksheet

Take advantage of Toad environment when working with Mongo shell.

Toad extends the functionality of Mongo shell, allowing you to execute various commands and queries and see data organized in grid.

To open a new Worksheet, click the Worksheet icon on Main Toolbar.

Once you open a Worksheet, you can use any command you would normally use in Mongo shell. Note that there is a slight difference when you use commands/queries:

  • Commands - Results of executed commands are displayed in DB Console View (Windows Menu | Show View | DB Console).
  • Queries ('find()' commands) - Results of executed queries are shown in a grid well-arranged grid.

Note: Toad usually executes commands in Mongo shell and queries via Mongo API. You can enforce executing of both commands and queries in Mongo shell by enabling the Use Mongo Shell Only option on Worksheet toolbar.

You can also take advantage of syntax highlighting. There are three options:

  • No brackets highlighting - no highlighting is used in Worksheet
  • Brackets highlighting - only braces are highlighted by their depth level
  • Brackets content highlighting - highlights code inside brackets
