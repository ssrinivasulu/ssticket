Connections

Find information about how to create a new database connection and manage connection properties.

Existing database connections

Stored Connections can be activated/deactivated and managed in:

  • Connect button | Manage Connections
  • Toad Menu | Preferences | Connections
  • Database Menu | Connections | Manage...

New database connection

To create a new database connection:

  1. Click the Connect button (additionally choose New Connection if you already have a stored Connection).

  2. Choose a database platform from the available choices (if you have only one database platform enabled, this dialog won't be displayed)

  3. In the New Database Connection dialog choose the Connection Method you want to use:
    • Manual Settings
    • Net Service Name (Oracle only)
    • Custom Connection String
  4. Enter the necessary connection details.
  5. Once you're done, you can try clicking the Test Connection button to see if the Connection is working.

  6. Save the Connection by clicking OK.

SSL/SSH secured connection

Using Manual Settings allows you to use SSL encrypted connection (PostgreSQL and MySQL only) or SSH tunneling.

SSL encryption

Check your database platform documentation for specific information about database settings, requirements and specifics for SSL encryption of the client/server communication.

Files needed to set up SSL encrypted database connection are:

  • CA file—Certificate Authority file
  • Cert File—server public key
  • Key File—server private key

SSH tunelling

SSH connection allows secure tunneling of your database connection through SSH server, which can run either on local machine together with your database, or on a remote server. To create an SSH tunneled connection, you must enter IP or FQDN in Host field and also credentials for an identified SSH User.

Note: Default port 22 used for SSH connections must be open for outgoing/incoming connections.

Other information

  • Passwords can be stored encrypted for any Connection.
  • Restore Active Connections on Startup option (enabled by default) restores your saved connections from previous session. See Restore Active Connections on Startup option in Toad Menu | Preferences... | Advanced.
  • In order to simplify tasks required for establishing a connection to your database, Toad sets default JDBC drivers for MySQL, Oracle and PostgreSQL (distributed with Toad) and searches for installed Oracle Clients automatically.
  • Default driver can be defined/changed in Toad Menu | Preferences | Databases.
