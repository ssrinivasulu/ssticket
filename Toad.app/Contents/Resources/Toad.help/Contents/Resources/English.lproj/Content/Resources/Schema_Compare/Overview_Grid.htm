Overview Grid

See differences between objects and mark them appropriately.

The grid displays all differences between objects of compared schemas and contains additional options to help you filter items and mark the differences as important and read/unread.

Note: You can hover over a row to see differences between two objects in a popup window.

Overview Grid Columns

Column Description
Source Schema Contains database objects located in source schema, for which differences have been found.
Action Displays action that should be executed in case of particular difference. Actions can be set via Comparison Area Toolbar.
Target Schema Contains database objects located in target schema, for which differences have been found.
Important Indicates whether a particular difference is marked as important.
Read Indicates whether a particular difference has been reviewed (read ) or not (unread ).
Excluded Differences marked as excluded can be hidden using Show/Hide Excluded Items button on Overview Grid Toolbar.

Overview Grid Toolbar

UI Element Option Description
Recompare All Compares Source and Target schemas again.
Export report Exports the result of Schema Compare to an HTML file.
Previous Unread Change Switches to the previous difference marked as unread ().
Next Unread Change Switches to the next difference marked as unread ().
Group by Groups the differences by selected criteria.
Filter Filters the differences.
Modified since When enabled, displays only objects, which have been modified since the set date.
Show/Hide Excluded Items Shows/Hides differences marked as Excluded.
Hide Empty Nodes Hides object type nodes with no differences.
Expand All

Expands all sub-items.
Collapse All Collapses all sub-items.
