Object Explorer

Easily access and view all of your database objects.

With Object Explorer you can comfortably navigate through objects of currently active (selected) database. Object Explorer displays objects of currently active database, which can be changed in Connections View.

Note: To quickly find a specific database object in any of your Connections, use Open Quickly function (ALT+COMMAND+O).

Object Explorer Toolbar

Option Description
New Object Creates a new database object, depending on currently selected item (e.g. creates a new Table when a Table is selected in Object Explorer.
Remove Object Deletes selected database object.
Options Contains options that can be used on currently selected object.
Collapse All

Collapses all items in Object Explorer.
Define Visible Categories Displays a dialog where you can choose which object categories should be shown in Object Explorer.
Link with Editor

When enabled, activating an Object Detail window highlights the appropriate object in Object Explorer and highlighting an object in Object Explorer activates the appropriate Object Detail window (if opened).
Display Opened Object in Common Describe Object Editor When enabled, you can open database objects in single Describe Object window simply by selecting them in Object Explorer.

Preferred and Active Schemas/Databases

In Oracle, databases you can set Schema as Preferred by right-clicking a Schema and choosing Add to Preferred. Such Schemas will be shown in their own category - Preferred Schemas.

In MySQL and PostgreSQL, you can set Schemas (PostgreSQL) or Databases (MySQL) as Active by right-clicking a Schema/Database and choosing Set Active Schema/Database. Active Schema/Database is the where SQL statements are executed by default.
