Sessions View

View your Oracle database sessions and their properties.

Note: This feature is available for Oracle databases only.

Note: Only sessions related to the currently logged user are available.

The Sessions View displays database sessions and their state. The display is updated continuously and all changing values are highlighted. The display can be filtered, for example, inactive sessions can be hidden. Sessions can be also terminated (Kill button)

To see current sessions,you need to select a connection first (via Select Connection dropdown menu)
If the currently logged user has SELECT privilege to appropriate tables, you can see the sessions information.
