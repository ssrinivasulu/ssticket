Describe Object

View and edit your database object details in one window.

Describe Object displays detailed information about specific database object and allows you to edit object properties (in most cases). It is useful to browse through details of many database objects, without opening an Object Detail window for every single of them.

To open a database object in Describe Object, make sure to enable Display Opened Object in Common Describe Object Editor option in Object Explorer toolbar. Then simply single-click an object in Object Explorer.

There can be only one instance of Describe Object opened per Connection.

TIP: You can quickly transfer from child to parent table by holding COMMAND and clicking on any value in Foreign Key column. You can recognize such values by hovering over them while holding COMMAND - they will be highlighted by blue color.
