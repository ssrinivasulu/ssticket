Copy Table

 

Copy Table wizard allows you to easily create a copy of any of your tables. It should be mainly used to create a backup of table data, since the copy only contains not null columns from the original table (the copy is made using CREATE TABLE .. AS SELECT statement).

To create a copy of a table in your database:

In the Object Explorer, right-click the table you want to copy and select Copy Table... option.

Copy Table wizard displays. Select the destination Schema and enter Table name. Additionally you can enter a comment.

In the next dialog check the columns that should be included in the table copy.

Click Finish to create the copy.
