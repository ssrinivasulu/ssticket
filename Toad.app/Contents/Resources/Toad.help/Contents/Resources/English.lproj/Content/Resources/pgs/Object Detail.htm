Object Detail

View and edit your database object details in separate windows.

Object Detail displays detailed information about specific database object and allows you to edit object properties (in most cases). Each object is opened in its separate Object Detail window.

To open a database object in Object Detail, simply double-click it in Object Explorer or select it and press COMMAND+ENTER.

One database object can be opened in one instance of Object Detail only. You can open as many Object Detail windows as you need.

TIP: You can quickly transfer from child to parent table by holding COMMAND and clicking on any value in Foreign Key column. You can recognize such values by hovering over them while holding COMMAND - they will be highlighted by blue color.
