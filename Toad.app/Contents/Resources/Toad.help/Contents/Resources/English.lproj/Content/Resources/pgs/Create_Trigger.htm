Create New Trigger

Use wizard to create a new trigger easily.

Right-click Triggers in Object Explorer and choose New Trigger.

The rest of the process is different depending on your database platform:

  • Oracle
  • PostgreSQL
  • MySQL

Oracle

In the displayed wizard define the Trigger Schema, Name and Type. Depending on the Type you selected, you will have to configure additional settings:

  • Table Trigger

  • View Trigger

  • Schema Trigger

  • Database Trigger

Once you're done, click Finish. The created trigger will be shown in SQL Worksheet.

PostgreSQL

In the displayed wizard define the Trigger Schema, Name, Type, Function and Constraint Options. Depending on the Type you selected, you will have to configure additional settings:

  • Table Trigger

  • View Trigger

MySQL

In the displayed wizard define the Trigger Schema, Name, Object and Fire/Event settings. Then click Finish and the newly created trigger will be shown in SQL Worksheet.
