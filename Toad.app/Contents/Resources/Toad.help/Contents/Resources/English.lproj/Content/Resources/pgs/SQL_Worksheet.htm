SQL Worksheet

Learn how to execute SQL statements and use other features connected with this editor.

The SQL Worksheet is used for unique executing of SQL statements during a database session.The results of executed SELECT statements can be seen in the Result Set section of Worksheet.

A new Worksheet can be opened via button on Main Toolbar.

Worksheet settings

Button Option Description
Execute All Statements

Contains three options:

  • Execute All - Executes all statements in the entire worksheet.
  • Execute To Current Position - Executes all statements in worksheet to the current cursor position.
  • Execute From Current Position - Executes all statements in worksheet from the current cursor position.
Execute Current Statement

Contains three options:

  • Execute - Executes statement on which line the cursor currently is (Shortcut: CTRL+ENTER)
  • Show Explain Plan - Displays query execution plan.
  • Execute SQL Statement with Profiler

TIP: You can also select (highlight) multiple statements and execute them using this button.
Cancel Query Cancels query that is being executed.
Commit Transaction Commits all DML changes to database.
Rollback Transaction Rolls back all DML changes to the
Select Connection Determines the connection used for executing queries.
Show Result Set Shows/hides the Result Set, where results of SELECT statements appear.
Horizontal Layout Switched to horizontal layout of Worksheet/Result Set.
Vertical Layout Switched to vertical layout of Worksheet/Result Set.
