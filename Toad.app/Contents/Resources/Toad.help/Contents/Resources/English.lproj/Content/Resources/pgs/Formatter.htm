Formatter

Find out about options for custom formatting of code.

Note: To use formatter, either click the Format option in context menuor press COMMAND+SHIFT+F.

By default, Toad uses its own formatter. However, you can also use your favorite formatter from Toad for Oracle by importing its configuration to Toad.

Configuring Formatter

Formatter options are available in Toad Menu | Preferences | Formatter.

Option Description
Formatter configuration
  • Default - uses default formatter
  • From XML file - uses custom formatter configuration imported from a file
Configuration file path The path to custom formatter configuration file.
Other Contains one option - Format script on load. When enabled, all database object represented by script will be formatted when opened.
