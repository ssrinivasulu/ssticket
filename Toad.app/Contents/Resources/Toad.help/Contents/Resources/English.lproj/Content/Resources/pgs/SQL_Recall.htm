SQL Recall

View and reuse executed SQL statements.

The SQL Recall dialog displays the last 100 executed SQL statements. The statements are grouped by connection and can be filtered by defining a searched expression.

In addition, messages can be pinned in the history dialog, so that they are not removed when a space is needed to enter new statements.

You can copy the entries of the history to the clipboard. E.g. to run them again in the SQL Worksheet.
