About Toad

Overview of what Toad can help you with.

Toad is a database development and management application for Oracle, PostgreSQL and MySQL databases designed to meet expectations and common practices of both professionals and geeks on their Macs.

Benefits

Note: The availability of some features is dependent on used database platform. For more information, check the specific features topics.

Toad allows you to:

  • Manage your database connections
  • View and explore the database structure, database objects and properties
  • Manage database objects - easily Add, Edit, Drop or Truncate objects
  • Run Stored Procedures and Functions with parameters
  • Manage large amount of data in grids; i.e. add, edit or remove records
  • Insert files into BLOB fields
  • Import data or Export data to various formats
  • Write and execute complex SQL statements comfortably by taking advantage of Code Completion, Syntax Check and on-the-fly Code Verification
  • Track executed SQL statements in SQL Recall View and see errors in SQL Problems View
  • Make your code more comprehensive using SQL code Formatter and Syntax Highlighting
  • Get detailed information about database objects - Explain Plan, SQL Profiler, Quick Doc, Dependencies, Oracle Parameters, etc.

Toad Views

  • Object Explorer — Allows you to browse through the objects of your database.
  • Object Detail — Displays the selected database object details and metadata.
  • Session View — Sessions related to the logged user.
  • SQL Recall — History of all user-executed SQL statements.
  • SQL Monitor — Log of all executed SQL statements, including internal ones.
  • DBMS Output View — Overview of all outputs from DBMS_OUTPUT.
  • Stored Procedure Editor — Editing Stored Procedures, Views, Triggers and compilation and running of PL/SQL.
  • Search Dialog — Browse the source code of stored procedures.
  • Quick Doc View— Shows the PL / Doc of the selected database object.
  • Outline View- Shows the logical structure of a database object.
  • Oracle Parameters View — Overview of the database and session parameters.
  • Dependencies View — Shows the selected database object dependencies.
  • Worksheet — Execution of any SQL statement.
