Preferences

Find out about available application settings and how to change them.

To define application preferences:

  • Go to Toad Menu | Preferences.

Preferences are divided into the following categories:

  • General
  • Appearance
  • Assist
  • Export
  • Databases
  • Connections
  • Formatter
  • Grid Options
  • Advanced

General

Option Description
Confirm any DROP, REVOKE and TRUNCATE statement When enabled, a dialog will ask you for confirmation every time you try to DROP, REVOKE or TRUNCATE something via User Interface.
Syntax Check Enables/Disables checking code syntax in SQL Worksheet/Stored Procedure Editor.
Case sensitive filtering Enabled/disables case sensitive filtering in Object Explorer.
Case sensitive dialogs

Determines object name behavior in various dialogs.

When enabled, object names entered by you will be saved exactly as entered. If the object name contains any special characters, the object name will be quoted.

When disabled, object names are converted to lowercase/uppercase automatically depending on database platform. If the object name contains any special character, you may encounter errors.
Query timeout settings You can set a time limit after which a currently processed query will be automatically canceled with an SQL Error: ORA-01013: user requested cancel of current operation.
SQL Editor Options

Here you can set the default SQL Worksheet and Result Set layout and also enable the option that hides Result Set when you are typing. Additionally, you can show/hide line numbers.
Syntax Coloring You can switch between several syntax coloring modes: Full (all objects and keywords are colored, Quick (only keywords are colored) and No coloring.
Object Explorer options Contains one option - Show database objects count, which shows the number of database objects next to object types in Object Explorer.
Preferred database platform Determines the default database platform used when creating a new Connection.
Bind Variables Contains one option - Replace bind variables... When enabled, replaces question mark character "?" with appropriate bind variable ":var" when inserting or executing a query in SQL Worksheet.
Prompt before quit When enabled, you need to confirm a dialog in order to close Toad.

Appearance

This section allows you to set custom Colors and Fonts in the application.

Assist

Option Description
Enable Content Assist Enables/disables Content Assist. Disable Content Assist in case of performance issues.
Auto Activation You can enable/disable Auto Activation of Content Assist. Triggers are characters after which Content Assist appears and delay determines the time between entering a trigger and Content Assist appearing.
Sorting and Filtering Switches between sorting Content Assist suggestions by dependencies/relevance or alphabetical order.
Insertion Enables/disables use of aliases in Content Assist statements.

Export

Option Description
Open file after execution Exported files will be automatically opened after export.
XML
Option Description
Encoding Sets the Encoding for exported XML files.
Naming

Sets name of root and row tags for exported XML files.
Options
  • Trim string values - when enabled, deletes all leading and trailing whitespace characters from strings
  • Include null fields - when enabled, null values are included in the exported XML file
  • Indent size - sets the number of spaces used for indentation
CDATA export selection Here you can choose which data types should be enclosed in CDATA tags in exported XML files.
HTML
Option Description
CSS Style You can choose CSS style of your exported HTML file from several different options.
CSV
Option Description
Options You can set custom Field and Text delimiters, Escape character and also define substitute for Null values. The other options allow you do Include column headers and enclose Numbers in quotations.

Databases

Oracle Database
Option Description
JDBC driver Sets the path to the used Oracle JDBC driver. The driver is needed to connect and work with an Oracle database.
Connection options Contains options which determine if all objects and preferred schemas should be loaded after connecting to the database.
Content Assist Here you can enable/disable public synonyms, built-in functions and packages suggestions in Content Assist.
Editor Options Compile package spec and body at once - When enabled, the package specification and body are compiled at once, otherwise they can be compiled separately.
Bind Variables Sets the bind variable format which replaces question marks when Replace bind variables... option is enabled in General.
PostgreSQL
Option Description
JDBC driver Sets the path to the used PostgreSQL JDBC driver. The driver is needed to connect and work with a PostgreSQL database.
Object Explorer Options

Contains two options:

  • Show system objects in the Object Explorer - shows/hides PostgreSQL system object (pg_*) in Object Explorer. You need to reconnect to your database to apply changes.
  • Load all objects after connect - determines if all database objects should be loaded after connecting to the database.
MySQL
Option Description
JDBC driver Sets the path to the used MySQL JDBC driver. The driver is needed to connect and work with a MySQL database.
Connection options Contains an option which determines if all database objects should be loaded after connecting to the database.
MongoDB
Option Description
MongoDB Console Once you locate Mongo shell (mongo.exe), you can use Worksheet to execute various Mongo shell commands/queries.
Drivers Toad Extension allows you to use multiple MongoDB drivers. Used driver can be specified in Connection Properties.
Tree Options You can set the number of Documents analyzed to estimate Document structure.
Search Options Limits the number of fetched Documents.
MongoDB Console Locate the mongo.exe file in order to execute mongo shell commands using Worksheet.
Format Date values in Grid When enabled, Date values are formatted according to the selected format in Grid Options.

Connections

This section displays all stored Connections. You can edit existing Connections, or create new ones.

Formatter

Option Description
Formatter configuration You can either use default formatter, or import a formatter file from an XML file.
Configuration file path Path to the formatter configuration file.
Other Contains one option - Format script on load. When enabled, all database object represented by script will be formatted when opened.

Grid Options

Option Description
Display (null) for NULL values When enabled, "(null)" is displayed for NULL values. When disabled, NULL values are displayed as blank cells.
Confirm record deletion When enabled, a confirmation dialog appears every time you try to delete a row.
Fetch limit Here you can enable/disable fetch limit (number of displayed rows in Data Tab and Result Set in SQL Worksheet) or change its number.
Spaces in strings Enables/disables deleting leading/trailing spaces in strings before executing INSERT/UPDATE statement.
Date and number format Contains various date and number format settings.
Tooltips
  • Show tooltips over results - when hovering over a value that is longer than its column, the value is fully displayed in a tooltip.
  • Show web links preview - when hovering over a web address in grid, the page preview is displayed in a tooltip.
BLOB Viewer

You can set a Viewer for every time of BLOB file format:

  • System - BLOB is opened in default system application associated with the particular file format
  • Text - BLOB is opened in plain text editor

Advanced

Option Description
Login timeout You can either use default formatter, or import a formatter file from an XML file.
Restore Active Connections on Startup When enabled, restored active Connections from last session on startup.
Open SQL Worksheet Opens a blank SQL Worksheet once database connection is established.
